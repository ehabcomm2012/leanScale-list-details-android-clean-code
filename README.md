# leanScale-list-details-android-clean-code
Android App for showing a list an details of games fetched from the apis :
https://api.rawg.io/api/games
https://api.rawg.io/api/games/[gameId]
Technologies used : 
 Kotlin,
 MVVM design pattern ,
 clean architecture,
 Hilt dependency injection ,
 navigation graph,
 Kotlin Coroutine,
 Retrofit2,
 Junit unit test ,...
